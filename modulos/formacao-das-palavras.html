O que é formação das palavras

Formação das palavras é o estudo dos processos pelos quais novas palavras são criadas a partir de outras já existentes.

Esses processos explicam grande parte do vocabulário da Língua Portuguesa.

Derivação

Derivação é o processo pelo qual uma palavra nova se forma a partir de outra, com a adição ou retirada de elementos.

Derivação prefixal

Ocorre com a adição de prefixo ao radical.

infeliz, refazer, desleal

Derivação sufixal

Ocorre com a adição de sufixo ao radical.

felizmente, jogador, leitura

Derivação prefixal e sufixal

Ocorre com a adição simultânea de prefixo e sufixo.

infelizmente, deslealdade

Derivação parassintética

Ocorre quando prefixo e sufixo são indispensáveis à formação da palavra.

✔ Correto: infelizmente

❌ Não existem: infelizmente → *felizmente / *infeliz

Derivação regressiva

Forma palavras por redução, geralmente criando substantivos a partir de verbos.

cantar → canto | cortar → corte

Derivação imprópria

Ocorre quando uma palavra muda de classe gramatical sem sofrer alteração em sua forma.

o bem, o não

Composição

Composição é o processo pelo qual duas ou mais palavras se unem para formar uma nova palavra.

Composição por justaposição

Os elementos se unem sem perda fonética.

guarda-chuva, passatempo, couve-flor

Composição por aglutinação

Ocorre fusão dos elementos, com perda fonética.

planalto (plano + alto), aguardente (água + ardente)

Outros processos de formação

Abreviação

Redução de palavras longas.

foto (fotografia), moto (motocicleta)

Sigla

Formação de palavras a partir das letras iniciais.

ONU, IBGE, USP

Neologismo

Criação de palavras novas para designar conceitos recentes.

streaming, influenciador
